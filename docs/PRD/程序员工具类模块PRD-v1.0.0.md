# 程序员工具类模块 PRD v1.0.0

## 📋 项目概述

### 🎯 项目背景
为了提升程序员的日常开发效率，需要建立一个集成化的工具集合模块。该模块整合了程序员在开发过程中经常使用的实用工具，包括JSON处理、文本比对和翻译等功能，通过纯前端技术实现，无需后端支持，提供便捷、快速的开发辅助服务。

### 💡 核心价值
- **效率提升**: 集成常用开发工具，减少工具切换成本
- **纯前端实现**: 无需后端支持，响应速度快，数据安全
- **一站式服务**: 多种工具集中管理，提升用户体验
- **开发友好**: 专为程序员设计，界面简洁，功能实用
- **免费使用**: 所有功能完全免费，降低开发成本

## 🚀 功能需求

### 1. JSON工具 (JSON Formatter & Validator)

#### 1.1 JSON格式化
- **输入区域**: 支持粘贴或输入原始JSON数据
- **格式化输出**: 自动美化JSON格式，增加缩进和换行
- **语法高亮**: 对JSON的key、value、数据类型进行颜色区分
- **折叠展开**: 支持JSON节点的折叠和展开操作
- **层级显示**: 清晰显示JSON的层级结构

#### 1.2 JSON验证
- **语法检查**: 实时检测JSON语法错误
- **错误提示**: 准确定位错误位置，提供详细错误信息
- **格式建议**: 对常见错误提供修复建议
- **状态指示**: 显示JSON有效性状态（有效/无效）

#### 1.3 JSON操作功能
- **压缩JSON**: 移除空格和换行，生成紧凑格式
- **复制功能**: 一键复制格式化后的JSON
- **清空重置**: 快速清空输入和输出区域
- **示例数据**: 提供常见JSON结构示例
- **文件导入**: 支持上传JSON文件进行处理

#### 1.4 高级功能
- **数据统计**: 显示JSON的节点数量、深度、大小等信息
- **历史记录**: 保存最近处理的JSON数据（本地存储）
- **文件下载**: 支持下载格式化后的JSON文件

### 2. 文本比对工具 (Text Diff Tool)

#### 2.1 文本对比功能
- **双栏输入**: 左右两个文本输入区域
- **实时对比**: 输入时实时显示差异
- **差异高亮**: 不同颜色标记增加、删除、修改的内容
- **行号显示**: 显示行号便于定位差异位置
- **滚动同步**: 左右两栏滚动位置同步

#### 2.2 对比模式
- **字符级对比**: 使用LCS算法实现精确到字符的差异对比
- **单词级对比**: 以单词为单位进行对比
- **行级对比**: 以行为单位进行对比
- **忽略选项**: 支持忽略空白字符、大小写等选项

#### 2.3 结果展示
- **统一视图**: 将两个文本合并显示，标记差异
- **并排视图**: 左右对照显示，清晰展示差异
- **差异统计**: 显示增加、删除、修改的行数或字符数
- **导航功能**: 快速跳转到上一个/下一个差异位置

#### 2.4 文件处理
- **文件上传**: 支持上传文本文件进行对比
- **编码支持**: 支持UTF-8、GBK等常见编码格式
- **大文件处理**: 优化大文件对比性能
- **结果导出**: 支持导出对比结果为HTML或文本格式

### 3. 百度翻译工具 (Baidu Translation Tool)

#### 3.1 翻译服务集成
- **百度翻译**: 集成百度翻译官方网站服务
- **iframe嵌入**: 通过iframe直接展示百度翻译完整界面
- **URL参数传递**: 自动将输入文本传递给百度翻译

#### 3.2 翻译功能
- **输入区域**: 简洁的文本输入框
- **即时显示**: 输入文本后立即在iframe中显示翻译界面
- **完整功能**: 用户可以直接使用百度翻译的所有功能
- **新窗口打开**: 支持在独立窗口中使用百度翻译

#### 3.3 用户体验
- **直观操作**: 直接在iframe中查看和使用翻译结果
- **官方服务**: 使用百度翻译官方网站，保证翻译质量和功能完整性
- **快捷访问**: 提供便捷的翻译工具访问入口

## 🔧 技术实现方案

### 1. 前端技术栈
- **框架**: Vue 3 + Composition API
- **UI组件**: Element Plus
- **文本对比**: 自研LCS算法实现精准字符级对比
- **翻译服务**: iframe嵌入百度翻译官方网站

### 2. 核心库选择
- **JSON处理**: 原生JSON + 自定义格式化和语法高亮
- **文本对比**: 自研LCS算法 + 自定义渲染
- **翻译服务**: iframe嵌入，无需额外API调用
- **本地存储**: localStorage存储历史记录

### 3. 性能优化
- **懒加载**: 工具模块按需加载
- **算法优化**: LCS算法优化大文本对比性能
- **内存管理**: 合理管理大文件处理时的内存使用
- **响应式优化**: 针对不同设备优化界面和交互

## 📱 用户体验流程

### 1. JSON工具使用流程
1. 用户进入JSON工具页面
2. 粘贴或输入JSON数据
3. 系统自动验证和格式化
4. 用户查看结果，进行复制或其他操作
5. 可选择保存到历史记录

### 2. 文本比对使用流程
1. 用户进入文本比对页面
2. 在左右两个输入框输入要对比的文本
3. 选择对比模式（字符级/单词级/行级）
4. 点击"开始比对"查看差异结果
5. 可切换并排视图/统一视图
6. 可导出对比结果为HTML文件

### 3. 百度翻译使用流程
1. 用户进入百度翻译工具页面
2. 在输入框中输入要翻译的文本
3. iframe自动加载百度翻译界面并传递文本
4. 用户直接在iframe中查看翻译结果
5. 可点击"新窗口打开"在独立窗口中使用
